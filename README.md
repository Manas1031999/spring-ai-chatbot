# 🤖 Spring AI Chatbot (Ollama + MCP)

A local AI chatbot built using **Spring Boot + Spring AI + Ollama**, with support for **MCP (Model Context Protocol)** to integrate external tools.

---

## 🔥 Features

- 🤖 AI Chatbot using **Spring AI**
- 🧠 Local LLM powered by **Ollama** (no API key required)
- 🔗 MCP Client-Server architecture support
- ⚡ Fast local inference (offline AI)
- 🛠 Extensible for tool integration (DB, APIs, etc.)

---

## 🧱 Tech Stack

- Java + Spring Boot
- Spring AI
- Ollama (Local LLM)
- MCP (Model Context Protocol)

---

## 🧠 Architecture

- User request → Spring Boot (Spring AI)
- Spring Boot → Ollama (LLM)
- Spring Boot → MCP Client
- MCP Client → MCP Server (tools/APIs)
